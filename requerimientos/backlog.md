# Product Backlog

Lista priorizada de funcionalidades del sistema.

| Prioridad | Historia de Usuario |
|----------|---------------------|
| 1 | Gestionar productos |
| 2 | Gestionar categorías |
| 3 | Gestionar clientes |
| 4 | Registrar venta |
| 5 | Controlar inventario |

## Descripción

1. Gestionar productos: registrar, consultar, actualizar y eliminar productos del inventario.

2. Gestionar categorías: registrar y consultar categorías para clasificar productos.

3. Gestionar clientes: registrar y consultar clientes que realizan compras.

4. Registrar venta: registrar ventas incluyendo cliente, productos vendidos y cálculo automático del total en Bolivianos.

5. Controlar inventario: actualizar automáticamente el stock después de cada venta.
