# Historias de Usuario

## HU1 – Registrar producto
Como administrador de la tienda,
quiero registrar nuevos productos en el sistema
para mantener actualizado el inventario.

## HU2 – Consultar productos
Como administrador de la tienda,
quiero visualizar la lista de productos disponibles
para conocer el inventario actual de la tienda.

## HU3 – Actualizar producto
Como administrador de la tienda,
quiero modificar la información de un producto
para corregir datos como precio, marca o cantidad.

## HU4 – Eliminar producto
Como administrador de la tienda,
quiero eliminar productos que ya no se venden
para mantener el sistema actualizado.

## HU5 – Registrar venta
Como administrador de la tienda,
quiero registrar la venta de un producto
para actualizar automáticamente el stock disponible.
