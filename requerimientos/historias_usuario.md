# Historias de Usuario Refinadas

## HU1 – Gestionar productos
Como administrador de la tienda  
quiero registrar, consultar, actualizar y eliminar productos  
para mantener el inventario actualizado.

### Criterios de aceptación

- El sistema debe permitir registrar productos con nombre, marca y precio en Bs.
- El sistema debe registrar la cantidad disponible en stock.
- El sistema debe permitir actualizar información del producto.
- El sistema debe permitir eliminar productos que ya no se comercialicen.

---

## HU2 – Gestionar categorías
Como administrador de la tienda  
quiero registrar y consultar categorías  
para clasificar los productos del inventario.

### Criterios de aceptación

- El sistema debe permitir registrar nuevas categorías.
- El sistema debe permitir actualizar la descripción de una categoría.
- El sistema debe mostrar las categorías disponibles.

---

## HU3 – Gestionar clientes
Como administrador de la tienda  
quiero registrar y consultar clientes  
para mantener un registro de las personas que realizan compras.

### Criterios de aceptación

- El sistema debe permitir registrar nombre y apellido del cliente.
- El sistema debe registrar la cédula de identidad.
- El sistema debe permitir registrar el teléfono del cliente.
- El sistema debe mostrar la lista de clientes registrados.

---

## HU4 – Registrar venta
Como administrador de la tienda  
quiero registrar ventas de productos  
para mantener un registro de las transacciones realizadas.

### Criterios de aceptación

- El sistema debe permitir seleccionar un cliente.
- El sistema debe permitir agregar productos a la venta.
- El sistema debe registrar la cantidad vendida.
- El sistema debe calcular automáticamente el total de la venta en Bs.

---

## HU5 – Controlar inventario
Como administrador de la tienda  
quiero que el sistema actualice automáticamente el stock  
para evitar vender productos no disponibles.

### Criterios de aceptación

- El sistema debe descontar la cantidad vendida del stock.
- El sistema no debe permitir vender productos sin stock disponible.
