# Historia del Problema (Sin Refinar)

Carlos Mendoza es el propietario de una tienda llamada TechZone Solutions dedicada a la venta de productos electrónicos para computadoras como laptops, teclados, mouse, tarjetas gráficas, memorias RAM y otros componentes.

Actualmente la tienda gestiona su inventario y ventas de forma manual utilizando cuadernos y hojas de cálculo simples. Esto ha provocado varios problemas como errores en el registro de productos, dificultad para conocer el inventario disponible y pérdida de información cuando se realizan ventas.

Además, el propietario necesita registrar información de los productos como nombre, marca, precio y cantidad disponible. También requiere poder agregar nuevos productos, modificar información existente, eliminar productos que ya no se venden y consultar el inventario de manera rápida.

Debido a estos problemas, el cliente solicita el desarrollo de un sistema web sencillo que permita gestionar el inventario de su tienda de manera organizada y eficiente.
