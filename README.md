# TechZone Inventory System

Sistema de gestión de inventario y ventas para una tienda de productos electrónicos de computadoras.

Este proyecto fue desarrollado como parte de la asignatura **Desarrollo de Sistemas**, aplicando metodologías ágiles para el análisis, diseño y desarrollo de software.

---

# Descripción del sistema

TechZone Solutions es una tienda dedicada a la venta de productos electrónicos para computadoras como teclados, mouse, tarjetas gráficas y otros componentes.

El sistema permite gestionar el inventario, registrar clientes y controlar las ventas realizadas en la tienda.

---

# Funcionalidades principales

El sistema permitirá:

- Gestionar productos del inventario
- Clasificar productos mediante categorías
- Registrar clientes
- Registrar ventas de productos
- Calcular automáticamente el total de ventas en Bolivianos (Bs)
- Actualizar automáticamente el stock de productos

---

# Modelo del sistema

El diseño del sistema incluye las siguientes entidades principales:

- Categoria
- Producto
- Cliente
- Venta
- DetalleVenta

Estas entidades permiten representar el inventario y las ventas realizadas en la tienda.
